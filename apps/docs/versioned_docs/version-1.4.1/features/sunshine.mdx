# <SunshineApiName />

**POST `v1/sunshines/text`**

## リクエスト
#### ヘッダー
**`accept`** string

#### リクエストボディ
**`text`** string **required** <br/>
最大500文字までを受け付けています。

### リクエスト例
<details>
  <summary>cURL</summary>
  <pre>
    <code>
      ```
        curl -X POST \
          --location 'https://api.pe-ace.net/v1/sunshines/text' \
          -H 'Content-Type: application/json' \
          -H "Authorization: Bearer <API_KEY>" \
          -d '{
          "text": "<TEXT>"
        }'
      ```
    </code>
  </pre>
</details>
